# Screenshots folder

After running the queries in VS Code with the Malloy extension,
take screenshots of each result and save them here with these filenames:

- rent_by_bedrooms.png
- parking_premium.png
- pet_pricing.png
- pnw_cities.png
- amenity_premiums.png

Then they will automatically appear in the README.md.

To take a screenshot in VS Code:
1. Run a query in queries.malloy or analysis.malloynb
2. The results appear as a table or chart below the query
3. Right-click the result area and save, or use Cmd+Shift+4 (Mac) to screenshot
4. Save the file with the correct name into this folder
